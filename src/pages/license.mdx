# 라이선스

## 텍스트 콘텐츠

각 페이지의 본문 텍스트는 [Touhou Patch Center](https://www.thpatch.net/) 등에서 가져온 자료를 한국어로 번역·정리한 것입니다.

- **라이선스**: [Creative Commons BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/)
- **저작권자**: 각 페이지 `:::info` 블록 참조

## 이미지 / 캐릭터 아트

`static/img/**` 이하 모든 캐릭터/게임 이미지의 저작권:

- **저작권자**: 上海アリス幻樂団 (Team Shanghai Alice) / ZUN
- **사용 범위**: 비상업적 팬 사용 (참고: thwiki, en.touhouwiki.net 가이드라인)
- **표기 책임자**: t-skt 위키 관리자

## 코드

이 사이트의 코드 (`src/`, `sidebars.ts`, `docusaurus.config.ts`)는 별도 라이선스를 따릅니다. [GitHub 레포](https://github.com/t-skt/twiki) 참조.

## 데이터 소스

본 위키의 데이터는 [t-skt/tdata](https://github.com/t-skt/tdata) (SSOT)에서 생성됩니다. tdata는 다음 어댑터에서 데이터를 수집합니다:

- **thpatch**: Touhou Patch Center (CC BY-SA 4.0)
- **legacy_v2**: thwiki 한국어 변환 (touhou_normalized v2)
- **manual**: t-skt 위키 관리자가 직접 보강한 데이터
